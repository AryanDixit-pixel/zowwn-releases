01How it works

Zowwn connects to a Telegram channel you own and control. Your files travel from your computer straight to your Telegram — they never pass through anyone else's servers.

A

Zowwn spots the USB drive and shows everything on it — every file, every folder.

C

Everything arrives in your own private channel — ready whenever you need it.

First time you open Zowwn, it walks you through connecting your own Telegram in under a minute. You stay in control of where your files go.

02Install it

Pick your system for the exact steps.

  1. Not sure which Mac you have? Click the menu (top-left) → About This Mac. If it says Apple chip (M1/M2/M3/M4), use the Apple Silicon download. If it says Intel, use the Intel one.
  2. Open the downloaded .dmg file. A window pops up showing the Zowwn icon.
  3. Drag Zowwn into the Applications folder shown in that window.
  4. Open Zowwn from your Applications (or Launchpad). The first time, macOS may ask "are you sure you want to open it?" — click Open. That's normal for a freshly downloaded app.
Zowwn is signed and notarized by Apple, so it opens cleanly — no scary warnings to work around.
  1. Run the downloaded Zowwn-Windows.exe by double-clicking it.
  2. If Windows shows "Windows protected your PC": click More info, then Run anyway. This appears because the app is new, not because anything's wrong.
  3. Follow the installer — choose where to install, then finish.
  4. Launch Zowwn from the Start menu or the desktop shortcut.
The "Windows protected your PC" message is standard for newer apps that haven't yet built up a download reputation with Microsoft. Clicking "More info → Run anyway" is safe here.
